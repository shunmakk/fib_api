## セットアップ
- リポジトリをクローン
  ```
  git clone https://github.com/yourusername/fib_api.git
  ```
- 依存関係をインストール
  ```
  npm install
  ```
- アプリケーションを起動する
  ```
  npm run start
  ```
- テストを実行する
  ```
  npm run test
  ```
